# Knobs To Implement

## Mixer
- External Input Volume – accept and mix an audio input channel.

## Modifiers
- Filter Emphasis – resonance control inside the ladder filter.
- Amount of Contour – envelope modulation depth into the filter cutoff.
- Filter Attack Time – attack stage for the filter envelope.
- Filter Decay Time – decay stage for the filter envelope.
- Filter Sustain Level – sustain level for the filter envelope.
- Loudness Attack Time – attack stage for the loudness contour.
- Loudness Decay Time – decay stage for the loudness contour.
- Loudness Sustain Level – sustain for the loudness contour.

## Output
- Phones Volume – independent gain stage for the headphone bus.
